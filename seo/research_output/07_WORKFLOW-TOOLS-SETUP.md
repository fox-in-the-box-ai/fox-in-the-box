# Content Workflow Tools Setup

## Option A: Trello (Simple & Free)

**Setup Time:** 30 min | **Cost:** $0 | **Best for:** Solo work

### Board Structure
- **Backlog**: All keywords awaiting work
- **Writing**: Currently being written
- **Review**: Waiting for final check
- **Published**: Live on site
- **Ranking**: Monitoring for ranking progress

### Card Fields
- Title: Keyword
- Description: Content outline + CTA
- Label: TIER 1/2/3, Month 1/2/3
- Due Date: Target publish date

## Option B: Airtable (More Powerful)

**Setup Time:** 1 hour | **Cost:** $0 (free tier) | **Best for:** Teams

### Tables
- Keywords (linked to Content)
- Content (linked to Backlinks)
- Backlinks (linked to Keywords)

### Views
- By Month (for planning)
- By Status (for workflow)
- By Tier (for priority)

## Option C: GitHub Projects (For Developers)

**Setup Time:** 20 min | **Cost:** $0 | **Best for:** Tech teams

### Workflow
1. Create GitHub Issues for each blog post
2. Use GitHub Projects board
3. Automate with GitHub Actions
