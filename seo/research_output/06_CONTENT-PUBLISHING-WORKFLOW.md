# Content Publishing Workflow

## Pre-Writing: Planning & Research
- [ ] Keyword assigned from master tracker
- [ ] Search intent verified (check SERP)
- [ ] Content outline drafted (3-5 main sections)
- [ ] Related keywords identified for natural mentions
- [ ] Internal links planned (3-5 target URLs)
- [ ] Sources/references gathered

## Writing
- [ ] First draft completed
- [ ] Word count: [target based on content type]
- [ ] H1 tag = main keyword (ONE ONLY)
- [ ] H2/H3 = related keywords naturally
- [ ] Images added (3-5 recommended)
- [ ] Alt text for all images
- [ ] Internal links: 3-5 to related pages
- [ ] External links: 2-3 to authoritative sources

## On-Page Optimization
- [ ] Meta title: [60 chars max, keyword-first]
- [ ] Meta description: [160 chars max, CTA included]
- [ ] URL slug: /keyword-style-lowercase-hyphens
- [ ] Headers: H1 → H2 → H3 (logical hierarchy)
- [ ] Images: Compressed, descriptive filenames
- [ ] Mobile-friendly: Check responsiveness
- [ ] Page speed: Target <3 seconds

## Publishing
- [ ] Upload to CMS
- [ ] Preview desktop + mobile
- [ ] All links tested
- [ ] Images display correctly
- [ ] Publish!

## Post-Publishing
- [ ] Update master tracker: Published_Date + URL
- [ ] Add to Google Search Console
- [ ] Share on Twitter/LinkedIn/Reddit
- [ ] Plan outreach for backlinks
- [ ] Set 30-day review reminder

## 30-Day Review
- [ ] Check GSC: Impressions? Ranking position?
- [ ] Check GA: Traffic to page?
- [ ] Check engagement: Time on page, scroll depth?
- [ ] If not ranking: Identify why, plan refresh
- [ ] If ranking top 10: Start backlink outreach
